# Notebook-Style Study Workspace

Mentora's main study experience is now organized as one focused `Estudio` workspace.

## Layout

- `Materiales`: manage, search, upload, and select study sources.
- `Tutor IA`: ask questions, study through chat, save useful answers as notes, and upload material from `+`.
- `Studio`: generate summaries, quizzes, flashcards, mind maps, APA help, tables, study guides, diagrams, and infographic outlines from selected materials.

The desktop layout keeps all three panels visible in one main canvas. Brand, active space switching, `Nuevo espacio`, progress, share, settings, profile, and sign-out live in the topbar instead of a repeated full-height sidebar. Tablet and mobile use collapsible panels and bottom tabs so the student can move between `Materiales`, `Chat`, and `Studio` without leaving the study context.

## Source Selection

Students explicitly choose which materials the chat and Studio tools use. If nothing is selected, Tutor IA answers in general study mode. If selected materials are ready, chat and tools use only those materials. If a selected material is still preparing, the UI says `Estoy preparando este material` and general chat remains available.

## Studio Tools

Studio outputs are persisted as generated artifacts:

- Resumen
- Quiz
- Flashcards
- Mapa mental
- Cita APA
- Tabla
- Guia de estudio
- Diagrama
- Infografia

Generated outputs can be copied or sent back into Tutor IA to continue studying.

## Notes

Assistant messages can be saved as notes. Notes are linked to the current study space and selected materials, appear in Studio, and can be renamed or deleted.

## Removed Fragmentation

`Inicio`, `Materiales`, `Tutor IA`, and `Practica` are no longer separate primary navigation destinations for the normal study flow. Their core behavior now lives inside `Estudio`.
