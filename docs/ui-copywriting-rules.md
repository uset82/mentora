# Mentora UI Copywriting Rules

Mentora copy should sound like a calm study assistant.

## Replacements

| Avoid | Use |
| --- | --- |
| Contexto vivo | Material usado |
| Sin documentos listos | Sin material |
| Cuando termine la ingesta | Estoy preparando tu material |
| Esperando fuentes | Preparando material |
| Fuentes | Materiales |
| Ruta sugerida | Siguiente paso |
| Ejecuta una herramienta | Crea tu primer recurso |
| chunks | contenido |
| processing_status | hide it |

## Tone

- Keep sentences short.
- Prefer verbs students understand: preguntar, subir, estudiar, practicar.
- Do not explain the backend.
- Do not use multiple CTAs with the same intent in the same panel.
